## Build
Get this repo:
```bash
hostname=<hostname>
sudo rm -rf /etc/nixos
sudo git clone https://github.com/Er-Hass/NixFrame /etc/nixos
sudo chown -R root:root /etc/nixos
```

Generate hardware config:
```bash
sudo nixos-generate-config
sudo cp /etc/nixos/hardware-configuration.nix /etc/nixos/hosts/ash/hardware.nix
```

Run flake:
```bash
nixos-rebuild switch --flake /etc/nixos#$hostname \
  --extra-experimental-features "nix-command flakes"
```


## Structure
```text
OSmium/
├─ flake.nix
├─ flake.lock                 # (created later)
├─ hosts/
│  └─ atlas/
│     ├─ hardware.nix        # generated by NixOS later
│     └─ configuration.nix   # host glue
├─ modules/
│  ├─ common/
│  │  ├─ system.nix          # shared system defaults
│  │  └─ home.nix            # shared HM bits (optional)
│  └─ roles/
│     ├─ workstation.nix     # dev desktop baseline
│     ├─ kde.nix             # KDE Wayland session
│     ├─ hyprland.nix        # alt session (later)
│     └─ laptop.nix          # power/touchpad (later)
├─ home/
│  └─ axon.nix               # HM user config
├─ secrets/                  # sops/age files later (gitignored)
├─ disks/                    # optional disko layouts
├─ overlays/                 # custom package overrides
│  └─ default.nix
├─ pkgs/                     # your own small packages
│  └─ default.nix
└─ README.md
```
